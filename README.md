# 📝 React Notes App

A clean and productive notes application built with React.
Create notes, manage daily tasks, organize priorities, and keep everything saved locally.

## ✨ Features

### 🗒️ Notes Management

* ➕ Add notes with title and description
* ✏️ Edit notes
* 🗑️ Delete notes
* 📌 Pin important notes
* 🔍 Search notes instantly
* 🚦 Priority levels: High / Medium / Low
* 🕒 Auto date & time on every note

### ✅ Daily Tasks

* ➕ Add daily tasks
* ☑️ Mark tasks as completed
* 🗑️ Delete tasks

### 💾 Storage

* Notes saved with Local Storage
* Data remains after refresh

## 🛠️ Tech Stack

* ⚛️ React
* 💛 JavaScript
* 🎨 Tailwind CSS
* 💾 Local Storage

## 🚀 Getting Started

```bash id="q0f3m7"
git clone https://github.com/pallavi-676/react-notes-app.git
cd react-notes-app
npm install
npm run dev
```
